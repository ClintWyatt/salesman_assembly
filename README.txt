To run the assembly program salesman.asm, follow the steps below.

Install visual studio 2019 community version. Watch this youtube video if you need help: https://www.youtube.com/watch?v=IsAoIqnNia4
Next, Install the irvine library ( a library for the assembly language) on your C drive by going to this web address: http://asmirvine.com/gettingStartedVS2017/index.htm
Next, double click on the "Project.sln" file in the following directory: C:\Irvine\Examples\Project32_VS2019. This step will open visual studio in the right environment.
Under the Solution Explorer tab, delete the "AddTwo.asm" by right clicking on the file and select remove.
Right click on "Project" in the solutions explorer tab and hover to add and new item.
Select the salesman.asm file and click on the button "add".
Last, click on the "Local Windows Debugger" button to run the program.

If you wish to exit the program before entering all the data, press enter -1 while the program is running on the terminal.
